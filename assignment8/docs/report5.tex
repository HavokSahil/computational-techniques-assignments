\section*{Problem Statement}
The objective is to \textbf{numerically evaluate} the integral
\[
\int_{0}^{\infty} \frac{1}{2 + 2x + x^2} \, dx
\]
using the \textbf{3-point Gauss-Laguerre quadrature} method.  

Since Gauss-Laguerre quadrature is designed for integrals of the form
\(\int_{0}^{\infty} e^{-x} g(x) \, dx\), a coordinate transformation is applied so that
\[
g(x) = \frac{e^{x}}{2 + 2x + x^2}.
\]

\section*{Methodology}

\subsection*{Gauss-Laguerre Quadrature (3-point)}
For a function of the form
\(\int_{0}^{\infty} e^{-x} f(x)\, dx\), the 3-point Gauss-Laguerre quadrature approximation is
\[
I \approx \sum_{i=0}^{2} w_i \, f(x_i),
\]
where \(x_i\) are the Laguerre nodes and \(w_i\) are the corresponding weights.

The 3-point nodes and weights used are:

\[
\begin{aligned}
x_0 &= 0.415775, & w_0 &= 0.711093 \\
x_1 &= 2.29428,  & w_1 &= 0.278518 \\
x_2 &= 6.28995,  & w_2 &= 0.0103893
\end{aligned}
\]

\subsection*{Integrand}
After transformation to include the weight function \(e^{-x}\), the integrand becomes
\[
f(x) = \frac{e^{x}}{2 + 2x + x^2}.
\]

\subsection*{Implementation Steps}
\begin{enumerate}
    \item Store the 3 Gauss-Laguerre nodes and weights.
    \item Define the integrand \(f(x) = e^x / (2 + 2x + x^2)\).
    \item Compute the weighted sum:
    \[
    I = \sum_{i=0}^{2} w_i \, f(x_i)
    \]
    \item Display the numerical value of the integral.
\end{enumerate}

\section*{Code}
\begin{lstlisting}[language=Matlab, caption={3-point Gauss-Laguerre Quadrature for $1/(2 + 2x + x^2)$}]
clc; clear all;

%% Program to compute the integral of 1/(2 + 2x + x^2)
% using 3-point Gauss-Laguerre quadrature
%
% Author: Sahil Raj
% Assignment 8 Problem 5

nodes = [
  0.415775;
  2.29428;
  6.28995;
];

weights = [
  0.711093;
  0.278518;
  0.0103893;
];

fn = @(x) exp(x) / (2 + 2 * x + power(x, 2));

I = 0.0;
for i = 1:3
  I = I + weights(i) * fn(nodes(i));
end

fprintf("The integral of the given function is: %f\n", I);
\end{lstlisting}

\section*{Results}
The numerical integration using 3-point Gauss-Laguerre quadrature yields:

\begin{itemize}
    \item \textbf{Integral Value:} \(I \approx 0.695201\)
\end{itemize}

\section*{Conclusion}
The 3-point Gauss-Laguerre quadrature provides an accurate approximation for
\[
\int_{0}^{\infty} \frac{1}{2 + 2x + x^2} \, dx.
\]

By applying the transformation to include the weight function \(e^{-x}\), this method efficiently evaluates integrals over \([0, \infty)\) with high accuracy using only three nodes.
